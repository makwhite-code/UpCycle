# UpCycle — Davidson's Fashion Exchange

UpCycle is a polished front-end prototype for a campus-only fashion marketplace concept. It lets Davidson students imagine buying, selling, renting, saving, messaging, and relisting clothing within a trusted student community.

## Live Demo Features

- Marketplace homepage and browse page
- Sample listings with filters and search
- Save/favorite items
- Cart and Venmo-style checkout flow
- Sell/rent listing form
- Messaging and offer flow
- Profile page with listings, purchases, and reviews
- Founder-style analytics dashboard
- Fully static GitHub Pages deployment

## Important Note

This version is intentionally a **portfolio demo**, not a production marketplace. It does **not** use Supabase, a database, or real login. The app opens directly so recruiters and reviewers can immediately explore the product.

Any listings or actions created while clicking through the site are temporary and only exist during that browser session.

## How to Host on GitHub Pages

1. Create a new public GitHub repository.
2. Upload `index.html`, `README.md`, and `.nojekyll`.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save and wait for GitHub to publish the live link.
